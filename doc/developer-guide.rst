.. _lsst.dm.OCPS.build:

Build and Test
==============

This is a pure python package. There is nothing to build except the documentation.

.. code-block:: bash

    create_topics OCPS
    setup -r .
    pytest -v  # to run tests
    package-docs clean; package-docs build  # to build the documentation
